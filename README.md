# 🔍 GitHub PR Review Agent

AI-powered CLI tool that reviews any public GitHub pull request instantly in your terminal.

## 🎯 What It Does

Paste a PR URL → Get a structured AI code review with bugs, security issues, and improvements — streamed in real-time.


## ✨ Features

- 🖥️ **Pure CLI** — Works entirely in terminal
- ⚡ **Real-time Streaming** — See AI review appear word-by-word
- 🔒 **Security Analysis** — Flags vulnerabilities and risky patterns
- 🐛 **Bug Detection** — Identifies logic errors and edge cases
- 💡 **Improvement Suggestions** — Code quality and performance tips
- 🆓 **100% Free** — Uses free tiers of Groq and GitHub API
